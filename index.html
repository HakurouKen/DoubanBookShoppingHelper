豆瓣购书助手

为网购图书添加豆瓣助力

Download as .zip Download as .tar.gz View on GitHub

Douban Book Shopping Helper (chrome extension)

解决一边在逛网上书店，一边去豆瓣看书评的来回切换的烦恼...

功能：

  • 为各大购物网站添加豆瓣的评分的显示以及豆瓣评论的快速通道
  • 增加各大购物网站的价格对比

目前支持网站：

  • 亚马逊中国（国际）
  • 京东商城
  • 当当
  • 北发图书网
  • china-pub
  • 苏宁易购

GreaseMonkey && TamperMonkey支持：脚本地址
